# JavaEnterpriseProject

Enterprise REST API built with Spring Boot, JPA, Flyway, and Kafka.

## Getting Started
- Run with H2: `mvn spring-boot:run -pl enterprise-rest-api "-Dspring-boot.run.profiles=h2"`
- Run with MySQL: `mvn spring-boot:run -pl enterprise-rest-api "-Dspring-boot.run.profiles=local"`
